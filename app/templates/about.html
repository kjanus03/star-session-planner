About Star Session Planner

Star Session Planner is my side project focused around my passion - stargazing. The website helps stargazing enthusiasts plan their sessions based on weather conditions, proximity of urban centers and astronomical events. For me planning a good stargazing session has always involved multiple things - checking the weather, finding a spot with a good view of the sky and knowing what was going to happen in the sky. That pushed me to create a website where all that information is readily available - no more need to check for the weather and meteor showers in two separate places.

Developed to make stargazing more accessible!

Credits

Star Session Planner wouldn't be possible without the following APIs and services:

  • OpenWeatherMap API - For real-time weather data.
  • OpenStreetMap API - For geographical data and mapping.
  • Open-Meteo API - For accurate weather predictions.
  • Open Elevation API - For elevation data.
